Play the video, "Bad Apple!!" on the Nintendo Switch

Original Video: [【東方】Bad Apple!!　ＰＶ【影絵】](https://www.nicovideo.jp/watch/sm8628149)

## Build
Make sure you have:

 * DevkitA64
 * libnx

You must specify Makefile.libnx.
```
make -f Makefile.libnx
```

## How to Run

1. Download `bad_apple.nro`.
2. Place `bad_apple.nro` in the following folder on your SD card:
   `/switch/bad_apple/bad_apple.nro`
3. Launch it from the hbmenu.
4. Enjoy!
